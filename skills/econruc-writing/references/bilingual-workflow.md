# Bilingual Economics Writing Workflow

Use when source material is Chinese, mixed Chinese-English, or the user asks for both languages.

## Principle

Do not translate sentence by sentence unless the user explicitly asks for literal translation. Rebuild the economics logic, then write in the target language.

## Chinese to English

1. Extract the economics skeleton:
   `问题 -> 制度背景 -> 数据 -> 识别 -> 结果 -> 机制 -> 贡献 -> 边界`
2. Remove policy-report filler:
   `高质量发展`, `重要意义`, `显著促进`, `赋能`, `协同推进`.
3. Convert vague verbs:
   - `影响`: estimate, affect, change, shift
   - `促进`: increase, raise, improve, expand
   - `抑制`: reduce, lower, dampen
   - `机制`: channel, mechanism, margin
4. Add missing economic units:
   percentage points, percent, standard deviations, yuan, firms, workers, households, counties, years.
5. Write English in active, compact sentences.

## English to Chinese

Use Chinese that economists naturally use in seminars and papers:

- `We study...` -> `本文考察...`
- `We exploit...` -> `本文利用...`
- `The estimates imply...` -> `估计结果意味着...`
- `The evidence is consistent with...` -> `证据与...机制相一致`
- `A remaining concern is...` -> `一个仍需注意的识别威胁是...`

## Dual output format

For journal text:

1. English draft.
2. 中文逻辑说明.
3. Term choices if technical terms are ambiguous.

For Chinese reports:

1. 中文正文.
2. English title/abstract/summary if needed.
3. Notes on causal language and identification limits.

## Common repairs

- Replace `significant` with the actual inference statement: statistically significant at what level, or economically meaningful.
- Replace `impact mechanism` with a specific channel.
- Replace `policy enlightenment` with `policy implication`, and state the margin.
- Replace `from the perspective of` with a direct question or mechanism.
