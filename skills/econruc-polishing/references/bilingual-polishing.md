# Bilingual Polishing

## English output from Chinese source

Process:

1. Extract claims and evidence from the Chinese draft.
2. Delete rhetorical filler.
3. Rebuild the paragraph around the economics message.
4. Translate terms by function, not dictionary.
5. Add a Chinese note explaining major logic changes.

## Chinese output from English source

Process:

1. Preserve economics precision.
2. Avoid awkward literal translation.
3. Use Chinese economics terminology that sounds like a paper, seminar, or report.
4. Keep causal hedging.

## Term choices

| Chinese | English |
|---|---|
| 识别策略 | identification strategy |
| 反事实 | counterfactual |
| 处理组/控制组 | treatment/control group |
| 政策冲击 | policy shock |
| 估计结果 | estimates |
| 经济含义 | economic magnitude or economic interpretation |
| 机制检验 | mechanism test or channel analysis |
| 异质性分析 | heterogeneity analysis |
| 稳健性检验 | robustness checks |
| 外部有效性 | external validity |

## Tone

English: compact, transparent, modest.

Chinese: accurate, less slogan-driven, suitable for economists.

When a Chinese draft contains government-report style language, convert it into research language rather than preserving rhetorical intensity.
